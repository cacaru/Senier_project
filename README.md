# Senier_project
졸업작품 백업


## 주제
유니티 3d의 물리엔진을 활용한 간단한 퍼즐게임 구현

### 팀
팀장 : 이준호 - 캐릭터 애니메이션, 움직임, 상호작용 구현    
팀원 : 안재용 - 큐브 상호작용, 큐브 애니메이션, 큐브 이동 구현    
팀원 : 권순호 - 마우스 조작, UI, UX(보조) 구현
